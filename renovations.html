A brief dive into my process

So that's not the absolute starting point, but the idea was to first i play a bit with some basic concepts, like here, and add more content and then refine step-by-step:

Cleaned a bit ...

I then added some more formatting, and really tried to go as step by step as possible, so i could feel more comfortable through developping a deeper knowledge.

Wrote some text, and after applying the school.css file, i was quite happy with the result, even if it kinda came by "magic".

The background color is powderblue.

Added Disqus which creates a comment/interactive section. Problem i faced was after a certain number of clicks the text color became white (as you can see on the picture).

After playing a bit with the "Color Scheme" setup on the Disqus website, i decided to not spend much time on this issue as it was clearly not a priority: i activated the "Light" mode instead of the "Auto" (the other one being "Dark").

I then added two sharing buttons.

The final touch needed to complete the entrance tests was to align them. Done by using the code found here.

Overall:

I really enjoyed the experience. Learning to use HTML, CSS and JS all in order to build my first blog (which was originally definitely not the objectif!) was great. I could do it again, this time much faster, so i'm happy with that. Add those facts to the mix:

  1. i built skills which increase my overall "coding understanding"
  2. i can use those skills to maintain a blog that will lead me to connect to a lot more people
... and you have a happy programmer!
